# K-Means Clustering Algorithm Implementation

## ðŸ“˜ Description and Overview:

This project implements the K-Means clustering algorithm from scratch in Java to group unlabeled data into clusters based on similarity. It demonstrates key concepts in unsupervised learning, including centroid initialization, distance calculation, and iterative cluster optimization.

The algorithm produces **six clusters**, representing the following trend categories:

- **Normal**
- **Cyclic**
- **Increasing Trend**
- **Decreasing Trend**
- **Upward Shift**
- **Downward Shift**

Each cluster represents a unique type of pattern that may appear in monitored or measured data.  
While not all six categories are visualized in this report, the clustering process accounts for all of them.
